# 🧠 Quiz_Time ⚡

Welcome to **Quiz Time**! A lightweight, engaging, and fully interactive web-based quiz application built from scratch using pure frontend technologies. Challenge yourself or others, test your knowledge, and get immediate feedback! 🚀

---

## ✨ Features

* **🎨 Sleek UI/UX:** Clean, modern design with smooth layouts.
* **⚡ Real-Time Scoring:** Your points update instantly as you answer questions.
* **📱 Fully Responsive:** Beautifully optimized to play on mobile, tablet, or desktop.
* **🚀 Lightweight & Fast:** Built entirely with vanilla JS—no heavy frameworks required!

---

## 🛠️ Tech Stack

* **🌐 HTML5:** Semantic structure and content layout.
* **🎨 CSS3:** Custom styling, modern layouts, and crisp responsiveness.
* **💛 JavaScript (ES6+):** Dynamic question rendering, interactive game logic, and state management.

---

## 🚀 How to Play

1. Open the application in your browser.
2. Read the question and choose the correct answer.
3. Track your score as you progress to the end! 🏆
